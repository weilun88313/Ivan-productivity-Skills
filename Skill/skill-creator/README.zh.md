# 技能创建器

> 使用最佳实践和模板创建新的 Claude Code 技能

## 概述

使用适当的结构、文档和配置文件来搭建新的 Claude Code 技能。确保一致性并遵循既定模式。

### 主要特性

- 📁 **结构化模板** - 预配置的目录结构
- 📝 **文档模板** - README、SKILL.md 和指南
- ⚙️ **配置文件** - 标准化的 skill.json 格式
- 🎨 **最佳实践** - 既定模式
- 🚀 **快速设置** - 几分钟内即可开始

## 快速开始

```bash
./create_skill.sh my-new-skill
```

## 技能结构

```
my-new-skill/
├── SKILL.md           # AI 指令
├── README.md          # 用户文档
├── skill.json         # 配置
├── scripts/           # 可执行脚本
├── references/        # 参考资料
└── examples/          # 使用示例
```
