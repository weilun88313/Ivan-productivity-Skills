# Skill 创建工具

> 使用最佳实践和模板创建新的 Claude Code 技能

## 概述

**Skill 创建工具** 帮助你构建具有适当结构、文档和配置文件的新 Claude Code 技能。它确保技能之间的一致性并遵循既定模式。

### 核心功能

- 📁 **结构化模板** - 预配置的目录结构
- 📝 **文档模板** - README、SKILL.md 和指南
- ⚙️ **配置文件** - 标准化的 skill.json 格式
- 🎨 **最佳实践** - 遵循既定模式
- 🚀 **快速设置** - 几分钟内开始使用

## 快速开始

```bash
# 运行 skill 创建工具
./create_skill.sh my-new-skill
```

## Skill 结构

```
my-new-skill/
├── SKILL.md           # AI 指令
├── README.md          # 用户文档
├── skill.json         # Skill 配置
├── scripts/           # 可执行脚本
├── references/        # 参考资料
└── examples/          # 使用示例
```
