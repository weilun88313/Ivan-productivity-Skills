# Twitter Post Writer

> 创建针对推荐算法优化的高互动 X (Twitter) 帖子

## 概述

通过应用经过验证的算法优化策略，撰写能够最大化覆盖面和互动的推文和线程。基于对 X 推荐系统的深入理解，本 Skill 指导你创建能够自然驱动点赞、转发和回复的内容。

## 核心功能

- 🎯 **算法优化写作** - 应用 X 算法优先考虑的排名因素
- 🌍 **英文输出** - 所有推文均使用专业英文生成
- 🖼️ **AI 图片生成** - 使用 Gemini 3 Pro Image 创建吸引人的视觉内容
- 📊 **互动策略** - 经过验证的提高互动率的技巧
- 🧵 **线程框架** - 多推文叙事的结构化模板
- ✅ **发布前检查清单** - 发布前验证内容质量
- 📚 **丰富示例** - 多种类别的真实案例模式

## 快速开始

直接让 Claude 帮你创建 Twitter 内容（可以用中文提问）：

```
"写一条关于 [主题] 的推文"
"创建一个解释 [概念] 的推文线程"
"优化这条推文以提高互动：[你的草稿]"
"生成一条带图片的推文，关于 [主题]"
```

Claude 会自动：
- 用英文撰写推文（如需要会自动翻译）
- 应用算法最佳实践
- 在适当时使用 Gemini 3 Pro Image 生成视觉内容
