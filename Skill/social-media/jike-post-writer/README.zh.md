# 即刻帖子写作

将碎片想法或话题转化为即刻风格的帖子，支持可选的 AI 配图生成。

## 功能特点

- **两种输入模式**：从零散想法整理成帖子，或围绕话题展开写作
- **口语化中文风格**：自然随意，像跟朋友聊天
- **灵活长度**：50-500字，根据内容自动调整
- **可选配图生成**：需要时通过 Gemini API 生成 1:1 正方形配图
- **一致的个人风格**：AI/科技、产品思维、创业、生活感悟

## 快速开始

安装 Skill 后，直接让 Claude 写即刻帖子：

- "帮我写个即刻帖子，关于今天用 AI 写代码的发现"
- "把这些想法整理成即刻帖子：[你的零散想法]"
- "写个即刻帖子，主题是产品思维，要配图"

## 输出

帖子保存为 `.md` 文件到工作目录，可直接复制粘贴到即刻。
