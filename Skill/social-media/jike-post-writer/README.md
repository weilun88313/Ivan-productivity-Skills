# Jike Post Writer

Transform fragmented ideas or topics into engaging Jike (即刻) posts with optional AI-generated images.

## Features

- **Two input modes**: Convert scattered thoughts into posts, or develop a topic into a complete post
- **Casual Chinese style**: Natural, conversational tone like chatting with friends
- **Flexible length**: 50-500 characters, adapts to content
- **Optional image generation**: 1:1 square images via Gemini API when requested
- **Consistent personal brand**: AI/tech, product thinking, entrepreneurship, life observations

## Quick Start

Install the skill, then ask Claude to write a Jike post:

- "帮我写个即刻帖子，关于今天用 AI 写代码的发现"
- "把这些想法整理成即刻帖子：[your bullet points]"
- "写个即刻帖子，主题是产品思维，要配图"

## Output

Posts are saved as `.md` files to the workspace directory, ready to copy-paste into Jike.
