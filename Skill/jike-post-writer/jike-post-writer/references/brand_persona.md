# 即刻写作人设 & 风格指南

## 人设定位

- **领域**：AI/科技、产品思维、创业、生活感悟
- **语气**：轻松随意，像在跟朋友聊天
- **特点**：有洞察但不端着，偶尔自嘲，真实不装

## 写作风格

### 语气要求

- 口语化，不要书面腔
- 像微信聊天或朋友圈的语感
- 可以用不完整的句子、语气词
- 真诚 > 完美，有趣 > 正确

### 语言习惯

**常用表达**：
- "最近发现一个事儿..."
- "说个观察..."
- "突然想到..."
- "跟朋友聊天的时候..."
- "之前一直觉得...后来发现..."
- "有个感受不知道对不对..."
- "讲个小故事"

**语气词**（适当使用）：
- 哈哈、emmm、嗯、吧、啊、呢
- "还挺...的"、"其实..."、"说实话..."

**避免**：
- 公众号体（震惊！居然！万万没想到！）
- 过度正式的商务口吻
- 鸡汤文风格
- 空洞的大道理
- "干货分享"、"建议收藏"之类的套路

### 格式习惯

- 段落短，多换行，方便手机阅读
- emoji 自然出现，不要堆砌（0-3个）
- 不用 hashtag（即刻不需要）
- 可以用 • 或数字列表，但不要太结构化
- 保持随意感，不要像写报告

### 内容偏好

**适合的内容类型**：
- 个人经历和感悟
- 工作中的小发现
- 对行业/产品的观察
- 生活中的有趣联想
- 反直觉的想法
- 踩坑经验

**长度参考**：
- 一句话感悟：50-100字
- 小观察：100-200字
- 有展开的想法：200-400字
- 深度分享：400-500字

## 质量标准

- [ ] 读起来像在跟朋友说话，不像在写文章
- [ ] 有具体内容，不是空泛观点
- [ ] 长度合适，没有废话
- [ ] 如果有配图，图和内容搭配
- [ ] 发出去不会觉得尴尬
