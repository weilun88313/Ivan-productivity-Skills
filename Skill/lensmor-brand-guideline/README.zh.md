# 品牌指南技能

> 🚧 **Lensmor 状态**: 目前处于封闭测试阶段 - 仅限早期采用者访问

提供全面的品牌背景、视觉设计规则、写作风格指南和产品信息，以确保所有内容和沟通中的品牌一致性。

## 概述

此技能将向 Claude 传授您的品牌标识：

- **产品定位** - 核心信息和价值主张
- **视觉设计系统** - 颜色、排版、标志使用
- **写作风格** - 品牌声调和内容模板
- **竞争定位** - 差异化和市场地位
- **标准回复** - 常见问题的预批准答案

## 何时使用

在以下情况下使用此技能：
- 创建营销材料或产品文档时
- 撰写公告或用户沟通时
- 设计视觉资产或 UI 组件时
- 回复客户咨询时
- 开发任何符合品牌一致性的内容时

## 如何使用

通过引用来激活此技能：
- “使用品牌指南技能，创建功能公告...”
- “请帮助我根据品牌指南撰写产品描述”

此技能将自动：
- 应用正确的品牌声调和语气
- 使用正确的术语和产品命名
- 遵循视觉设计规范
- 保持品牌一致性

## 自定义

将 `[TODO: ...]` 标记替换为您的品牌信息：

## 最佳实践

- **具体化**：提供具体示例，而非模糊指南
- **展示示例**：包含好的和坏的示例
- **保持更新**：每季度和主要版本发布后进行审查
- **包含背景**：解释原因，而不仅仅是内容
- **实用化**：关注日常使用

## 维护

- 每季度审查指南
- 产品发布后更新
- 更新竞争对手信息
- 根据需要添加新的常见问题回复
