# 品牌指南技能

提供全面的品牌背景、视觉设计规则、写作风格指南和产品信息，以确保所有内容和沟通中的品牌一致性。

## 概述

此技能教授 Claude 您的品牌标识：

- **产品定位** - 核心信息和价值主张
- **视觉设计系统** - 颜色、排版、标志使用
- **写作风格** - 品牌声音和内容模板
- **竞争定位** - 差异化和市场地位
- **标准回复** - 常见问题的预批准答案

## 何时使用

在以下情况下使用此技能：
- 创建营销材料或产品文档
- 撰写公告或用户沟通
- 设计视觉资产或 UI 组件
- 回复客户咨询
- 开发任何与品牌一致的内容

## 如何使用

通过引用此技能来激活它：
- “使用品牌指南技能，创建一份功能发布公告……”
- “请根据品牌指南帮我撰写一份产品描述”

此技能会自动：
- 应用正确的品牌声音和语调
- 使用恰当的术语和产品命名
- 遵循视觉设计规范
- 保持品牌一致性

## 最佳实践

- **具体明确**：提供具体示例，而非模糊指南
- **展示示例**：包含好的和坏的示例
- **保持更新**：每季度和主要版本发布后进行审查
- **包含上下文**：解释原因，而不仅仅是内容
- **实用性强**：侧重于日常使用

## 维护

- 每季度审查指南
- 产品发布后进行更新
- 更新竞争对手信息
- 根据需要添加新的常见问题回复
