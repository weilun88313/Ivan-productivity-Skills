# Lensmor 品牌规范 Skill

一个全面的 Lensmor 品牌规范技能，提供品牌上下文、视觉设计规则、写作风格指南和产品信息，确保所有内容和沟通的品牌一致性。

## 这个 Skill 的作用

这个技能教 Claude 关于 Lensmor 的品牌标识，包括：

- **产品定位和价值主张** - 关于 Lensmor 做什么、为谁服务的核心信息
- **视觉设计系统** - 颜色、字体、Logo 使用和视觉元素
- **写作风格和语气** - 品牌声音、内容模板和消息指南
- **竞争定位** - 如何相对于竞争对手定位 Lensmor
- **标准回复** - 对常见问题的预批准答案

## 何时使用此 Skill

在以下情况使用此技能：
- 创建营销材料或产品文档
- 撰写功能公告或用户沟通
- 设计视觉资产或 UI 组件
- 回应客户问题或支持咨询
- 开发需要保持品牌一致性的内容

## 如何使用此 Skill

1. **激活技能**，在请求中提及它：
   - "使用品牌规范技能，为...创建功能公告"
   - "帮我遵循 Lensmor 品牌规范写产品描述"

2. **技能将自动**：
   - 对所有书面内容应用品牌声音和语气
   - 使用正确的术语和产品命名
   - 遵循视觉设计规范
   - 保持与既定品牌标准的一致性

## 自定义此 Skill

此技能附带需要填写实际品牌信息的模板文件：

### 核心文件

- **`SKILL.md`** - 主要技能指令和指南
  - 产品概述和定位
  - 视觉设计规范（颜色、字体、Logo）
  - 写作风格和语气指南
  - 内容模板和示例

### 资源文件（在 `resources/` 目录中）

- **`product-details.md`** - 详细的产品规格、功能和用户画像
- **`visual-assets.md`** - 完整的设计系统，包含颜色、字体、间距和资源
- **`competitor-comparison.md`** - 竞争定位和对比卡片
- **`faq-responses.md`** - 对常见问题的标准回复

### 填写你的品牌信息

在整个文件中查找 `[TODO: ...]` 标记，并用你的实际品牌信息替换：

1. **从 `SKILL.md` 开始**：
   - 添加产品描述和核心功能
   - 定义调色板（hex 颜色代码）
   - 指定字体（字体家族和用法）
   - 记录品牌声音属性
   - 添加写作示例（好的 vs 避免的）

2. **填写 `resources/product-details.md`**：
   - 详细的功能描述
   - 用户画像
   - 技术规格
   - 定价信息

3. **完成 `resources/visual-assets.md`**：
   - 添加 Logo 文件路径
   - 使用 CSS 变量定义完整的颜色系统
   - 记录间距、边框、阴影
   - 链接到设计模板

4. **更新 `resources/competitor-comparison.md`**：
   - 列出主要竞争对手
   - 定义关键差异化因素
   - 添加定位声明
   - 创建异议处理回应

5. **自定义 `resources/faq-responses.md`**：
   - 填写实际的 FAQ 回复
   - 调整问题以匹配常见客户询问
   - 添加产品特定信息

## 有效品牌规范的技巧

1. **具体明确**：模糊的指南会导致不一致的结果。提供具体示例。

2. **展示，不只是说明**：包含好的和坏的示例来说明你的观点。

3. **保持更新**：随着品牌的发展，定期审查和更新品牌规范。

4. **包含上下文**：解释为什么做出某些选择，而不仅仅是它们是什么。

5. **使其实用**：专注于实际日常会使用的指南。

2. Claude 将加载技能并将其应用于你的对话
3. 技能在整个对话中保持活跃
