# Keyword Research

Discover and prioritize keywords for SEO and GEO content strategies. Generates actionable article topic recommendations based on search intent analysis and competitive assessment.

## Key Features

- **Seed keyword brainstorming** across core terms, problems, solutions, comparisons, and questions
- **Intent classification**: Informational, Commercial, Transactional
- **Qualitative competition assessment** (honest about limitations without SEO tools)
- **GEO opportunity identification** for AI-citation potential
- **Topic clustering** for building topical authority

## Quick Start

```
Research keywords for [topic/product/service]
```

```
Find keyword opportunities for [industry] targeting [audience]
```

The skill outputs a prioritized table of article topics with target keywords, intent classification, and competition estimates.

## Ahrefs Integration

**This skill uses Ahrefs MCP** to provide real keyword data:
- ✅ Accurate monthly search volume
- ✅ Keyword Difficulty (KD) scores 0-100
- ✅ Traffic potential estimates
- ✅ SERP analysis

**Requirements**: Ahrefs API v3 access (included in Standard plan $249/mo with 500 free queries/month)
